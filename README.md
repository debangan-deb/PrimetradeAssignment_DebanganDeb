A Mini E-Commerce Website 
<!-- Tech Stack -->
<!-- Frontend -->

- React + Vite

- TailwindCSS

- Axios for API calls

<!-- Backend -->

- Node.js / Express

- MySQL

- JWT Authentication

- bcryptjs for password hashing

<!-- Core Features -->
- Authentication

- User Signup

- User Login

- Password hashing with bcrypt

- JWT token generation

- Protected routes (requires token)

- User Profile

- View logged-in user profile

- Update profile information

- Tasks Module (CRUD)

- Create a product

- View all products

- Update a product

Delete a task

Tasks are user-specific (protected endpoints)
